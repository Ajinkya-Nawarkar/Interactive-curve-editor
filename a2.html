Help

Commands:

  • line x0 y0 x1 y1 Create a line between the two integer points
  • poly x0 y0 ... xn-1 yn-1 Create a polygon at the given integer points
  • color r g b Set the current drawing color; values between 0..255
  • clear Reset the canvas

Optional Commands:

  • circle x0 y0 r Create a circle centered at the integer point with the given radius
  • curve type x0 y0 ... xn-1 yn-1 Create an open curve of type ("bezier" or "chaikin") with the given control polygon
  • closed type x0 y0 ... xn-1 yn-1 Create a closed curve of type ("bezier" or "chaikin") with the given control polygon
